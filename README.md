# Enterprise Network Security Assessment Lab

## Overview

This project simulates a cybersecurity assessment of a small financial organization.

The objective was to:
- analyze network architecture
- identify security risks
- perform asset discovery
- evaluate vulnerabilities
- recommend security improvements

## Tools Used

- Draw.io
- Nmap
- Wireshark
- Python

## Methodology

The assessment follows principles from:
- NIST Cybersecurity Framework
- CIS Controls
- MITRE ATT&CK

## Project Structure

1. Network Architecture
2. Asset Discovery
3. Risk Assessment
4. Security Recommendations
5. Framework Mapping
